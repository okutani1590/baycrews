# baycrews

# 📘 開発環境レギュレーション（README）

本ドキュメントは特集ページ制作における実装ルールです。
必ず内容を確認の上、実装してください。

------------------------------------------------------------------------

# ■ 全体方針

-   CMS環境（独自の）での実装を前提とする
-   外注様のテスト環境で確認　問題がなければ　弊社にてCMS反映を行う
-   シンプルで保守性の高いコードを優先する

------------------------------------------------------------------------

# ■ CSS

## ▼ 基本ルール

### 1. グローバル指定は禁止

    p { ... }
    body { ... }
    html { ... }

### 2. スコープを限定する

    .box-wrapper .title { ... }
    .box-wrapper p { ... }

### 3. 既存CSSは原則変更禁止

-   例外：フェード時間など軽微な調整のみ可

------------------------------------------------------------------------

## ▼ 使用ファイル

-   style.css または style.scss

------------------------------------------------------------------------

## ▼ 納品ルール

-   制作ファイル一式納品
-   コンパイル済みCSSで納品

------------------------------------------------------------------------

## ⚠️ 編集禁止

base.css は絶対に編集しないこと

------------------------------------------------------------------------

# ■ JavaScript

## ▼ 基本方針

-   軽微なアニメーションのみ使用
-   script.js を使用

------------------------------------------------------------------------

------------------------------------------------------------------------

# ■ クレジット表示

## ① CMS表示

HTML：

    <div id="item1" class="itemCredit"></div>

JS：

    <script>
      $(function () {
        $("#sec02").appendTo("#item1");
        $("#sec03").appendTo("#item2");
      });
    </script>

------------------------------------------------------------------------

## ② テキスト表示

    <div id="item1" class="itemCredit">
      <ul>
        <li><a href="" target="_blank">PANTS ¥00,000 intax</a></li>
      </ul>
    </div>
